Directory for vue components [[#^33cfe5]]

This is referenced in [[nextcloud news vue readme#Steps]]

  • Vue components go in js/vue_components.
  • That folder does not get referenced in the gulpfile or webpack config
    • Rather, components in that folder should be referenced in js/app/VueComponents.js
    • That is the entry point of the webpack config
    • The output for the webpack config is js/webpacked_vue_components.js
