Creating Side Tools to Make Obsidian an Org-Mode Replacement

General idea

Writing notes in a plain text editor, I should be able to:

  1. Define tasks
  2. Assign a due date, start date, and completion date for that task

Using a plain command-line tool (so that it can become a text editor plugin), I should be able to:

  1. See an "agenda" list of tasks, sorted by the upcoming days of the week, with a list of tasks that are due on each day
  2. Print out a Gantt chart showing when tasks were done

Existing work

It would be great if this was compatible with existing Obsidian plugins that already do some of this.

Obsidian Tasks

  • Allows you to query for tasks by due date
  • Allows you to attach start/end/due dates to a task

Obsidian Dataview

  • Specify words on the page as data
  • Query for that later

  • Has a javascript API. Can that be used externally?
