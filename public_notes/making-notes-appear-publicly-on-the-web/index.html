Making an Obsidian Notebook Show Up on the Web

Goal here is to do something like what the Obsidian Publish feature does, but doable on your local machine for static sites made with things like [[eleventy]].

Objectives

This project should be approachable in small pieces. Right now you can take an obsidian file and render it as HTML, although none of the Obsidian-specific syntax will do anything. Piece by piece, that could be implemented.

Some eventual goals:

  • Double-bracket links go to a page matching that name
  • Show pages that link to a given page
  • Show pages that the current page links to
  • If a double-bracket links is created, and no such page exists, show a default page that shows which other pages link to that one.
  • Show tasks as a checkbox
  • Embed things created by dataview
